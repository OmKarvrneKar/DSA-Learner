# DSA-Learner

An interactive application designed to help learners understand and practice Data Structures and Algorithms (DSA) through visualizations and practical examples.

## 🚀 Features
- Intuitive interface for algorithm visualization
- Built with React and Vite for blazing fast performance
- Interactive learning modules

## 📦 Tech Stack
- React
- Vite
- JavaScript / HTML / CSS

## 🛠️ Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
